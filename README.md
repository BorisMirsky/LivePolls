# LivePolls
Система опросов и голосований в реальном времени.

Аналог Slido или Mentimeter. 


*****

### Как пользоваться:

* склонировать

* dotnet restore

* run

*****

### backend и тестовая страница работают, можно погонять:

* Swagger: http://localhost:5063/swagger/index.html       

* тестовая страница: http://localhost:5063/test-hub.html         


*****

### Особенности реализации:

- 'Авторизация' по имени.

- Один юзер в одном опросе голосует один раз.

- Пользователи создают опросы и голосуют в реальном времени. Результаты обновляются на экране мгновенно.

- Возможно много подключений одновременно.

- Вовзможно будет фронтенд на vue.js, пока что только тестовая страница (LivePolls.Web/wwwroot/test-hub.html) с чистым js
