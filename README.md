# LivePolls

Система опросов и голосований в реальном времени.

Примерный аналог приложений `Slido` или `Mentimeter`. 




### Как пользоваться:

* Склонировать

* Установить зависимости `dotnet restore`

* Запустить проект



### backend и тестовая страница

* Swagger: http://localhost:5063/swagger/index.html       

* тестовая страница: http://localhost:5063/test-hub.html         




### Особенности реализации

- 'Авторизация' по имени.

- Один юзер в одном опросе голосует один раз.

- Возможно много подключений одновременно.

- Пользователи создают опросы и голосуют в реальном времени. Результаты обновляются на экране мгновенно.

- База данных на `SQLite` находится в `backend/LivePolls.DataAccess/Database/`

- Используется `Entity Framework`.

- Тестовая страница реализована на чистом `js` внутри `html`.

- Позже будет, возможно, полноценный фронтенд на `Vue.js`.
